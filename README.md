# Unstable Diffraction

A browser-based glitch art and psychedelic image processing suite. No dependencies, no build step — just a single HTML file.

**[▶ Launch App](https://kaib89.github.io/Unstable-Diffraction/)**

## Effects

| Effect | Category | Description |
|--------|----------|-------------|
| **Turing Patterns** | Reaction | Difference-of-Gaussians reaction-diffusion with linked, luma, and RGB split modes |
| **Pixel Sort** | Glitch | Sort pixel spans by luminance, hue, or individual channels |
| **Channel Shift** | Glitch | Independent X/Y offset per RGB channel |
| **Chromatic Aberration** | Optical | Radial, horizontal, or vertical color fringing |
| **Feedback Loop** | Optical | Iterative zoom-rotate-blend for fractal echo effects |
| **Halftone** | Optical | Dot/square/diamond patterns in BW, RGB, or CMY |
| **Posterize** | Color | Reduce color levels per channel |
| **Invert** | Color | Invert all or individual channels |

## Features

- **Stackable effects** — add multiple instances, reorder via drag & drop
- **Per-effect toggle** — enable/disable without removing
- **RGB Split mode** — independent activator/inhibitor/strength per color channel (Turing Patterns)
- **Undo/Redo** — full history with Ctrl+Z / Ctrl+Shift+Z
- **Export** — save result as PNG
- **Zero dependencies** — runs entirely in the browser, no server needed

## Usage

1. Open the app via [GitHub Pages](https://kaib89.github.io/Unstable-Diffraction/) or open `index.html` locally
2. Drop an image or click to import
3. Add effects from the sidebar
4. Adjust parameters, reorder, toggle effects
5. Export your creation as PNG
